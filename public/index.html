How to use:

            

                This tool ensures security by generating unique passwords using SHA256 without storing them on your
                machine or any server. It achieves this by using and combining the provided information for each login.
            

            

                - Site or App:
                

                    Specify the website or app name for password use. Adopt a consistent pattern for easy recall; for
                    example, simplify "https://www.google.com" to "google.com" by excluding subdomains.
                

            

            

                - LoginId:
                

                    Including your login ID is essential for the tool to tailor the generated password specifically to
                    your login credentials. This personalized approach enhances security by incorporating your unique
                    identifier into the SHA256-based password generation process, ensuring a password that is both
                    robust and individualized for your account.
                

            

            

                - Master Password:
                
This is the master password known only to you, used for generating unique passwords.

            

            

                - Password Length:
                
Specify the desired length for the login credential password.

            

            

                - Password Variant:
                

                    Create unique passwords for the same credential using this option.
                

            

            

                - Save Button:
                

                    Store entered information locally for easy retrieval of your password.
